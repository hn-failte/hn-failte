# Windows命令控制台默认编码问题

有遇到过在Windows命令行中乱码的问题吗？今天的主题就是这个。解决命令控制台默认编码问题。

## 1、默认编码

Windows在中国的默认编码为936，即gb2312

## 2、查看当前编码

chcp

## 3、修改编码

chcp [code]
```bash
# 切换为utf8
chcp 65001
```

## 修改cmd的默认编码

注册表修改：

路径`HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Command Processor`下，新建字符串，命名`autorun`，值为`chcp 65001`
